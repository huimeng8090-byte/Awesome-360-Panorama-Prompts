## 🍌 简介

本仓库收集了来自 **[绘梦拾光](https://www.diysq.com)** 的 GPT Image 2.0 360° 全景图生成提示词。

每个提示词均采用标准化 JSON 结构，包含：
- `图像元数据`：投影方式、宽高比、分辨率
- `场景构图`：地点、相机位置、高度、天气
- `视觉元素`：前景、中景、背景、整体氛围
- `渲染风格`：光照、调色、质感
- `约束条件`：必须保留 / 必须避免
- `负面提示词`：排除不需要的内容

---

## 📰 使用说明

1. 找到你想要的场景案例
2. 复制 `提示词` 代码块中的 JSON 内容
3. 粘贴到 **GPT Image 2.0**（ChatGPT 图像生成）中直接使用
4. 生成结果为 **2:1 横幅比例**，适合 360° 全景查看器加载

> [!NOTE]
> 所有提示词均针对**等距柱状投影（Equirectangular Projection）**优化，生成的图片可直接用于 VR 全景查看器或 Pannellum 等工具。

## 📊 提示词结构说明

所有提示词均采用标准化 JSON 结构，字段含义如下：

| 字段 | 说明 |
|------|------|
| `图像元数据` | 定义图像类型、投影方式、宽高比（必须为 2:1） |
| `场景构图` | 地点、相机位置与高度、视野范围、天气、景深 |
| `视觉元素` | 前景/中景/背景的具体内容和动态描述 |
| `渲染风格` | 光照方案、色彩调色、质感要求 |
| `全景约束强化` | 确保左右无缝拼接的专项约束 |
| `约束条件` | 必须保留的核心元素 + 必须避免的内容 |
| `负面提示词` | 排除项列表，防止生成不需要的内容 |
